## Environment
- Composer (2.0.13)
- PHP (7.4.27)
- Redis
- MySql(8.0.26)
- Node (14.16.1)

## Setup
- Install library
    - composer install
- Migrate database
    - php artisan migrate
- Run server
    - php artisan serve
- Run server socket
    - cd server && node index
